Berkeley AutoLab meshpy Documentation
=====================================
Welcome to the documentation for the Berkeley AutoLab's `meshpy` module!
This module is enables basic operations with 3-dimensional
triangular meshes such as reading, writing, editing, rendering,
and computation of stable poses.
The `meshpy` module depends on `numpy`_, `scipy`_, and
`matplotlib`_, which you can install with pip.

.. _numpy: http://www.numpy.org/
.. _scipy: https://www.scipy/org/
.. _matplotlib: http://www.matplotlib.org/
