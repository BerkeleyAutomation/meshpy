Stable Poses¶

Discrete set of poses for which a triangular mesh may rest on an infinite planar worksurface when dropped with an initial orientation uniformly at random.

StablePose¶

class meshpy.StablePose(p, r, x0, stp_id=-1)¶

p¶

float – Probability associated with this stable pose.

r¶

numpy.ndarray of :obj`numpy.ndarray` of float – 3x3 rotation matrix that rotates the mesh into the stable pose from standardized coordinates.
